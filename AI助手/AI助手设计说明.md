# AI 助手设计说明

## 助手名称

**智联（ZhiLian）**

---

## 形象设计

| 特性 | 描述 |
|-----|------|
| 外观风格 | 极简科技风，液态金属质感 |
| 颜色变化 | 可随场景变换颜色（工作-蓝色，休闲-紫色，娱乐-橙色） |
| 形态 | 3D 悬浮头像，非实体人形 |
| 表情 | 可根据对话内容呈现表情变化（微笑、思考、惊讶等） |
| 声音 | 自然人声，支持多种风格切换 |

## 核心能力

| 能力 | 说明 |
|-----|------|
| 日程主动提醒 | 天气、出行、事务、作息智能推送 |
| 聊天语义理解 | 捕捉对话关键词，主动推荐相关服务 |
| 一站式事务处理 | 发消息、设提醒、规划路线、生活便民服务一键完成 |
| 个性化语音人设 | 可切换风格（亲切/专业/活泼），适配不同使用场景 |
| 学习进化 | 长期使用越了解用户习惯，提供更精准服务 |

## 互动视频脚本

| 时间 | 场景 | 画面 | 对话内容 |
|-----|------|------|---------|
| 0:00-0:08 | 问候场景 | 用户拿起手机，智联头像浮现，带有微笑表情 | 智联："早上好，今天天气晴朗，气温 22-28 度，适合户外活动" |
| 0:08-0:18 | 日程提醒 | 日历提醒弹出，智联头像变为思考表情 | 智联："您下午 3 点有个产品会议，需要我帮您准备相关资料吗？" |
| 0:18-0:28 | 导航场景 | 用户说"我要去公司"，地图打开，智联头像变为专业表情 | 用户："我要去公司"<br>智联："好的，已为您规划最快路线，预计 25 分钟到达，当前路况良好" |
| 0:28-0:40 | 情感互动 | 用户表现出疲惫，智联头像变为关切表情，推荐音乐 | 智联："您看起来有点累，要不要听些轻松的音乐？我为您准备了您喜欢的轻音乐列表" |
| 0:40-0:45 | 结尾 | 智联头像微笑，渐变为品牌标识 | 智联："有什么需要随时叫我" |

## 语音风格

| 风格 | 适用场景 | 特点 |
|-----|---------|------|
| 亲切模式 | 日常、休闲 | 温暖柔和，语速适中 |
| 专业模式 | 工作、商务 | 简洁干练，语速稍快 |
| 活泼模式 | 娱乐、聚会 | 轻松愉快，富有活力 |
