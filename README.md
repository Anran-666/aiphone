# AI Phone——共创下一代智能手机

## 项目简介

本项目以"AI Phone——共创下一代智能手机"为主题，融合柔性显示、人工智能、无感交互等前沿技术，设计一款面向未来的新一代智能终端。

## 使用说明

### 1. 设计文档

详细设计方案请查看 `项目设计文档_V2.0.md`

### 2. 外观设计图

- 设计说明与 AI 绘图 Prompt 请查看 `外观设计图/设计图说明.md`

- 交互体验视频脚本请查看 `演示视频/交互体验视频脚本.md`

## 创新亮点

- **形态创新**：可变式柔性机身，三种形态切换
- **交互创新**：无感多维交互，一步直达
- **智能创新**：AI 主动预判服务
- **AI 助手创新**：智联（ZhiLian）深度集成
